This is Step 6: Adding a Tailored Illustration

Suppose that you now want to embellish your letter with a graphic that changes according to the gender of the recipient. The Mail Merge Helper in Word is very useful for inserting variable text but inserting variable graphics is a bit trickier, requiring some additional steps.

The simplest way to insure that your graphics that will appear as expected using the Mail Merge function is to create a separate file for each graphic. For this demonstration, open a new document and create a balloon or another simple shape with Word's drawing tools and color it pink. Hold down the shift key as you select each part of the drawing (left)
and join the parts by clicking on Draw on the Drawing toolbar and choosing Group.
The handles will move to the outer border of the balloon, because it's now one picture.


Repeat the process to create a blue balloon. Now copy the grouped drawing of the pink balloon and paste it as a picture into a separate document or file. Use the Paste Special item under Edit on the Main menu (Left).

The Paste Special dialog box asks how you want to paste the object. Choose Pictureand uncheck the Float over text box. Then click OK.


A correctly pasted picture will have a black border when selected. The pink balloon should be the only thing in the file. Paste the blue balloon in the same way into its own file. For the purposes of Tailoring and Mail Merge, these files, each containing an individual graphic, are called Source documents. They should be saved with individual names, in a separate folder on the desktop. Follow this procedure for all graphics whether they are Excel graphs, Illustrator vector drawings, clip art or drawing objects you create in Word.

We now are ready to link the graphics to the Main document, so turn on your View Field Codes button. We will be placing our embedded code into a text box to insure that the graphic remains in the same place on the page as multiple documents are merged. Insert a text box where you would like the graphic to appear. Adjust the size to accommodate your graphic. With your cursor placed inside the text box, click Insert Word Field and choose If...Then...Else. In the Word Field If dialog box, the Field name is again gender, the comparison is again equal and the value is F. Type "Pink balloon goes here" (without the quotes) into the first Insert this text box and click OK. The Word field is now visible in the text box. Select the words inside the first pair of quotes (Pink balloon goes here). They are acting as a placemarker for the picture of the balloon.
With the placemarker words selected, go to File under Insert on the Main menu.

The resulting dialog box will ask you to navigate to the file to be inserted, in this case, the picture of the pink balloon. Checking the Link to File box is crucial. Now you can click Insert.


Inside the text box, (left) you will see that the place-marker has been replaced with embedded code. Between the quotes are another set of curly braces with the linked field name Includetext and a path from your computer’s hard drive to the pink balloon Source Document. When you turn off the View Field Codes button, and turn on the View Merged Data button, you should see the pink balloon when viewing the record of a female. Turn off the View Merged Data button, turn on the View Field Codes, place your cursor after the last curly brace of that embedded code and insert a Word field for the next variable: the blue balloon. Follow the same procedure you followed for the pink balloon, but substitute an M for the F in the Word Field If dialog box. Remember not to leave a space between the two Word fields. Press the Update button after adding Word fields. Turn off the View Field Codes, button, turn on the View Merged Data button and the gender-appropriate graphic should appear as you use the Record arrows to move from record to record.

As we saw in Step 5, Word fields allow us to view all the variations of text at one time in the embedded code. In the case of graphics, however, what we see in the code are the links to the individual Source documents. You can understand now why placing the folder containing your graphic files on the desktop and to keep the path short is beneficial.
