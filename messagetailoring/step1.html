This is Step 1: Creating a Sample Database

Launch Microsoft Word. A new document will open.
Create a table for your database: Under Table on the main menu select Insert Table.* The Insert Table dialog box will appear (Left). Fill in the number of rows and columns. We will use 6 columns and 4 rows for our demonstration.

Using the Insert Table dialogue to begin a sample database.

Here is an example of a database for you to copy. The top row is the "header" row containing all the field names of your variables: Fname (first name), Lname (last name), Address, etc. The next three lines are records; each made up of the values for one individual. These values are placed beneath the appropriate field name.

Fname Lname Address City, State, Zip Gender Race This is the Header Row
Martha Sample 1 Treeline Dr. Balooga, MA 12345 F Cauc Record 1
George Example 2 Whisper Lane Florence, FL 23456 M AA Record 2
Fred Newbee 3 Spree Lane Catawah, MI 34567 M Cauc Record 3

The sample table you will create. It's a database!

Create a new folder called "Tailoring Demonstration" on the desktop for the documents you will create in this exercise. Save your database to this desktop folder, naming it "Tailoring Database" and then close the database document. Now, we move on to Creating a Sample Master Letter.
