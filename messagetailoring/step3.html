This is Step 3: Linking the Main Document to the Database

Now link your database to your Main document: under 2: Data source, click Get Data and select Open Data Source... from the drop-down list (Left.)

Merge Helper to link your letter to your database.

A list of files will appear. Navigate to your demonstration database in your folder on the desktop and click Open. Your main document is now linked to your database.

Now a warning box will appear saying: "Word found no merge fields…" Don't panic! Click on the Edit Main document button (Above) which will return you to the Main document where you will compose your letter.

But first save this Main document to your folder on the desktop with the name Tailored Letter Master. Do not close it, as we are ready to insert our first tailored text.

If you've satisfactorily linked the document to the database, we can go to Inserting Merge Fields into the Main Document.
